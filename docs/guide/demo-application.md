# Demo Application

The official [PayRex for Laravel demo](https://payrex-laravel-demo-main-89yckp.free.laravel.cloud/login) is a live application showcasing every feature of the package. It covers payment intents, checkout sessions, billing statements, customers, webhooks, PayRex Elements, etc. You can find the source code on [GitHub](https://github.com/whoami15/payrex-laravel-demo).

The database is reset every midnight.
